Cách thực thi chương trình:
1. Chạy file cài đặt setup.exe trong thư mục QuanLyNhaSach_setup
2. Dùng SQL Server để Attach file QuanLyNhaSach.mdf trong thư mục database
3. Chạy shortcut chương trình QuanLyNhaSach đã cài đặt trên Desktop.

Ngoài ra, nếu gặp lỗi phiên bản SQL Server thì có thể sử dụng file script.sql hoặc file QuanLyNhaSach.sql để tạo database.
